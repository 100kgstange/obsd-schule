If, else if, else

public class IfElseExample {

    public static void main(String[] args) {

        int number = -10;

        if (number > 0) {
            System.out.println("Zahl ist positiv");
        } else if (number < 0) {
            System.out.println("Zahl ist negativ"); // Ausgabe
        } else {
            System.out.println("Zahl ist gleich null");
        }
    }
}

While-Schleife:

Die while-Schleife ist eine der grundlegenden Kontrollstrukturen in Java, die es ermöglicht, einen bestimmten Codeblock so lange auszuführen, wie eine bestimmte Bedingung wahr ist.

Hier ist eine kurze Erklärung, wie die while-Schleife funktioniert:

  1. Bedingung: Die Bedingung ist ein Ausdruck, der wahr oder falsch ist. Solange die Bedingung wahr ist, wird der Codeblock innerhalb der Schleife wiederholt ausgeführt. Wenn die Bedingung falsch wird, wird die Schleife beendet, und die Ausführung des Programms setzt nach der Schleife fort.

  2. Codeblock: Der Codeblock innerhalb der Schleife enthält die Anweisungen, die wiederholt ausgeführt werden sollen, solange die Bedingung wahr ist.

Hier ist ein einfaches Beispiel, das eine while-Schleife verwendet, um die Zahlen von 1 bis 5 auszugeben:

public class Main {
    public static void main(String[] args) {
        int i = 1; // Initialisierung der Zählervariable

        while (i <= 5) { // Bedingung: solange i kleiner oder gleich 5 ist
            System.out.println(i); // Ausgabe der aktuellen Zahl
            i++; // Inkrementierung der Zählervariable
        }

        System.out.println("Schleife beendet!");
    }
}

For Loop

  1. Beispiel 1: Zählschleife von 1 bis 5
  • Eine for-Schleife wird verwendet, um von 0 bis einschließlich 5 zu zählen (for (int i = 0; i <= 5; i++)).
  • In jedem Schleifenzyklus wird der aktuelle Wert von i auf der Konsole ausgegeben (System.out.println(i)).
  • Die Schleife beginnt bei 0 und geht bis einschließlich 5, da der Testausdruck i <= 5 ist. Nach jedem Schleifenzyklus wird i um 1 erhöht (i++).
  1. Beispiel 2: Iteration über ein Array
  • Ein Array mit den Werten {10, 20, 30, 40, 50} wird erstellt: int[] numbers = {10, 20, 30, 40, 50};.
  • Eine for-Schleife wird verwendet, um über die Elemente des Arrays zu iterieren (for (int i = 0; i < numbers.length; i++)).
  • In jedem Schleifenzyklus wird die Nachricht "Element " + i + ": " + numbers[i] auf der Konsole ausgegeben. Dies gibt den Index i und den Wert des entsprechenden Elements im Array an.
  • Die Schleife läuft so lange, wie i kleiner als die Länge des Arrays ist (i < numbers.length). Nach jedem Schleifenzyklus wird i um 1 erhöht (i++).
public class ForLoopExample {

    public static void main(String[] args) {

        // Beispiel 1: Cycle from 1 to 5

        for (int i = 0; i <= 5; i++) { // i++ adds +1 in cycle
            System.out.println(i);
        }

        // Beispiel 2:

        int[] numbers = {10, 20, 30, 40, 50};

        for (int i = 0; i < numbers.length; i++) {

            System.out.println("Element " + i + ": " + numbers[i]);
        }
    }
}

Switch

Dieser Java-Code zeigt die Verwendung der switch-Anweisung zur bedingten Ausführung von Codeblöcken, basierend auf dem Wert einer Variablen. Hier ist eine Erklärung für jeden Teil des Codes:

  1. Eine Klasse namens SwitchExample wird erstellt.

  2. In der main-Methode wird eine Variable LandAuswahl mit dem Wert 2 initialisiert.

  3. Die switch-Anweisung wird verwendet, um verschiedene Fälle basierend auf dem Wert von LandAuswahl zu behandeln.

    • Fall 1 (case 1): Wenn LandAuswahl den Wert 1 hat, wird "USA" auf der Konsole ausgegeben. Da LandAuswahl den Wert 2 hat, wird dieser Fall übersprungen.

    • Fall 2 (case 2): Wenn LandAuswahl den Wert 2 hat, wird "Germany" auf der Konsole ausgegeben. In diesem Fall wird der zugehörige Codeblock ausgeführt, und die Ausgabe ist "Germany".

    • Fall 3 (case 3): Wenn LandAuswahl den Wert 3 hat, wird "China" auf der Konsole ausgegeben. Da es keinen break-Statement nach diesem Fall gibt, würde der Code nach der Ausführung des Falls 3 fortgesetzt werden. Dies ist jedoch nicht relevant, da der vorherige Fall (Fall 2) bereits ausgeführt wurde.

    • Default (default): Wenn keiner der vorherigen Fälle mit dem Wert von LandAuswahl übereinstimmt, wird der default-Fall ausgeführt, der "Wrong" auf der Konsole ausgibt.

  4. In diesem speziellen Beispiel hat die Variable LandAuswahl den Wert 2, und daher wird nur der Codeblock unter case 2 ausgeführt. Die Ausgabe auf der Konsole lautet daher "Germany".

Wichtig ist, dass nach jedem case-Block das break-Statement verwendet wird, um die Ausführung der switch-Anweisung zu beenden und zu verhindern, dass die nachfolgenden Fälle ebenfalls ausgeführt werden.

public class SwitchExample {

    public static void main(String[] args) {

        int LandAuswahl = 2;
  
        switch (LandAuswahl) {

            case 1:
                System.out.println("USA");
                break;

            case 2:
                System.out.println("Germany"); // Ausgabe Germany
                break;

            case 3:
                System.out.println("China");

            default:
                System.out.println("Wrong");
        }
    }
}

  2. Scanner schließen: Der Scanner sollte nach der Verwendung geschlossen werden, um Ressourcen freizugeben.
